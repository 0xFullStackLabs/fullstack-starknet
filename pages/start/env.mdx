# 设置开发环境



    
## 安装前置依赖

* [Python 3.9](https://www.python.org/downloads/release/python-390/) 
* [Rust](https://www.rust-lang.org/tools/install) 

## 安装Starknet命令行工具
接下来我们来安装Starknet CLI，也就是命令行工具。

* 设置虚拟环境

推荐使用Python虚拟环境来隔离项目依赖和全局依赖。你可以使用`venv`或者`pyenv`。

使用 `venv`:

```
# code blcok
python3.9 -m venv ~/cairo_venv
source ~/cairo_venv/bin/activate
```

创建激活虚拟环境之后，你应该在命令行看见`(cairo_venv)`，表示虚拟环境已经激活。
```
(cairo_venv) satoshi@nakamoto-notebook %
```

* 为你的系统安装必要的依赖

1. Ubuntu
```
sudo apt install -y libgmp3-dev
```

2. Mac
```
brew install gmp
```

完成安装以上系统相关的依赖后，继续安装Starknet CLI所需的Python包：
```
pip install ecdsa fastecdsa sympy
```
这些包提供了密码和数学功能

* 安装Starknet CLI:

1. 卸载之前版本的`cairo-lang`
```
pip uninstall cairo-lang
```

2. 安装最新版本的`cairo-lang`
```
pip install cairo-lang
```
3. 安装完之后，检查安装的版本，确保是[最新的](https://github.com/starkware-libs/cairo-lang/releases)。
```
starknet --version
```

## 安装Cairo编译器

1. 克隆Cairo仓库并设置
```
cd ~/
git clone https://github.com/starkware-libs/cairo/ .cairo
cd .cairo/
git checkout tags/v1.0.0-alpha.6
cargo build --all --release
```

2. 把Cairo可执行程序加入PATH环境变量

更新 `.bashrc` 或者 `.zshrc`
```
export PATH="$HOME/.cairo/target/release:$PATH"
```

重新打开一个命令行检查版本
```
cairo-compile --version
```

本地开发环境到此已经设置完成。
