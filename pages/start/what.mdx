# 什么是Starknet?

Starknet是以太坊上的一个Layer2解决方案，基于zk-STARKs的零知识证明技术。这项技术使它拥有更高的TPS、更有表达力的智能合约以及更低的Gas费。

Starknet包含以下几个组件：
1. Starknet智能合约：Starknet上的智能合约是用Cairo语言编写，它是专为zk-STARK证明设计的图灵完备的语言。
2. Starknet节点：节点负责处理和验证Starknet上的交易和证明，并将他们提交到以太坊。
3. Starknet桥：负责以太坊和Starknet之间的通信，包括充提币以及智能合约交互。
